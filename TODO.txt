FEATURES:

    Dikunt should be able to make ascii art of a set of predefined images.
    This will allow users to not find a url every time they need to make a
    reaction.

    Dikunt should be able to give breaking news on request. The command could
    for example be "dikunt: breaking"

    Dikunt should be able to display a list of features and a help string for
    each of the features when requested.

    A user should choose the width of his/her picture when running ascii art.
